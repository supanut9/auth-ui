# auth-ui

Phase-1 hosted login and consent UI for the platform.

## Stack

- React
- Vite
- TypeScript

## Local Defaults

- auth-ui: `http://localhost:3005`
- auth-server: `http://localhost:8050`

## Responsibilities

- render login page
- render consent page
- render OTP page
- render logout and error pages
- call `auth-server` flow endpoints
